<h1>CRUD API with JWT Authentication (Express + MongoDB)</h1>

<p>A RESTful CRUD API with signup and login has built using Express, and MongoDB (Mongoose).</p>
<p>This project supports authorization with full CRUD operations, pagination, filtering by status, and bulk inserts using modern ES6 syntax and async/await.</p>

<h3>Installation:</h3>
Clone the repository:

<strong>git clone https://github.com/tanjiyaJuthi/crud-with-jwt-authentication-using-express-and-mongoose</strong>

<h3> Move to project folder</h3>
cd todo-crud-api(folder name where you downloaded the app)

<h3>Install dependencies:</h3>
npm install

<h3>Start MongoDB: Make sure MongoDB is running locally.</h3>

mongodb://127.0.0.1:27017

<h3>Run the server:</h3>
<strong>npm start</strong>
Server will run at:
http://localhost:3000>

<h3>Features:</h3>
<ul>
    <li>Create, Read, Update, Delete todos and user using JWT authentication</li>
    <li>Pagination support</li>
    <li>Mongoose schema validation</li>
    <li>Router validation</li>
    <li>Populate relational data</li>
    <li>Clean RESTful API design</li>
</ul>
